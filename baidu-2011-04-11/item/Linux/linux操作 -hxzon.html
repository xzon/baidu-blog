1.远程登陆MySQL
mysql -h主机地址 -u用户名 －p用户密码
注意：主机地址不要加入端口号。
2.查看MySQL版本号 包括其它信息
mysql> status
3.编码问题
查看mysql支持的编码：
show character set;
查看各端编码
show variables like '%character%', 出现
mysql> show variables like '%character%';
4.启动MySQL
[root@neudiawww ~]# /etc/init.d/mysqld
Usage: /etc/init.d/mysqld {start|stop|status|condrestart|restart}
5.卸载
如果是 rpm 安装的话 可以用 rpm -e mysql-server
若果是rhel 的 可以用 yum remove mysql
6.安装MySQL
rpm -ivh MySQL-server-5.0.67-0.i386.rpm
显示

启动，一下两条命令都不成功。
sevice mysql start（注意大小写，这里必须全小写）
