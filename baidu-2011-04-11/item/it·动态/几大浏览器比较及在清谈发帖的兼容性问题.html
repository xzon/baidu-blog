几大浏览器比较及在清谈发帖的兼容性问题

其实这个话题没太多可说的。大家大部分都用 IE ，随便提几句吧。

kid 在做网页，所以手头几大浏览器统统都装了，好方便测试自己的
作品。顺便也就测试一下，到底哪个好使。这个注定是许多行家里手
都争吵不休的话题。kid 也就是从主观上随便谈谈吧。

第一大浏览器 : IE：

目前是主流，大概咱这里也有 85% 以上的XDJM 在用它，
优点是
1。 Windows 自带，不需要任何安装及设置，
2。 兼容性也近乎最好。目前大部分网页都为 IE 优化，测试也常常只测试 IE，
自然是最适合绝大部分网页的浏览器了。和清谈没有什么冲突。

缺点是：
1。占资源大，开多少窗口就占多少内存，几十个窗口可以吃光内存。
2。安全性不好，很容易被安装插件，中个木马，被修改一下主页，乱丢广告和 popup 窗口什么的。很烦。
3。功能差。
     开十个窗口，有一个崩溃了所有 IE 全死，想找出自己刚才看什么，没门。
     用 IE 存储某个网页，想带图片存下来就很难，尤其是带几十张图片的。比如懒虫 MM 那个航母的文章，我存了几十次都不行。
     没有多 Tab 功能。
     没有过滤飘来飘去广告的功能。等等等等。

太多了,kid 从来不用 IE 。除非去升级Windows。

2。用 IE 核心的包装浏览器，代表作比如 GreenBrowser 和 Maxthon。
优点：IE 的优点一个都不少，外加上

1。弥补 IE 占资源大的问题，在多开很多窗口时，内存占用没那么厉害。
2。弥补 IE 安全性问题，一般都可以设置禁止修改主页，禁止安装 ActiveX，
防止了木马和恶意 Javascript 的侵入。
3。功能强，基于 IE 核心，有时候问题还不能避免，但已经好的多了：
     即使 IE 核心崩溃，浏览器关闭，但它们常常带有保存上次网页的功能，还能找出来刚才死在哪里了。
     自带了多 Tab 功能，可以不用开上几十个 IE 窗口了。
     很多都带了 鼠标拖拽功能，很方便开新窗口。
     很多都自己集成了 Google 搜索，非常方便。
     放大页面都很方便，对高分辨率屏幕支持非常好。

所以 kid 为了兼容性，需要 IE 的时候绝对用这些浏览器。它们拥有 IE 的
全套优点并且弥补了绝大部分缺点，它们才是下文浏览器的竞争对手，
单纯的 IE，差的太远了。

优点：
1。轻快， 打开网页速度快，比 IE 及 GreenBrowser 都强。
2。安全性好。比 IE 及 GreenBrowser 都强。
3。多页面。一点都不比 GreenBrowser 差，比 IE 强。（需要插件）
4。支持鼠标拖拽，一点都不比 GreenBrowser 差，比 IE 强。（需要插件）
5。支持方便的下载 ，一点都不比 IE 及 Flashget 逊色（需要插件）
6。支持杀掉广告和漂浮窗口，一点都不比 GreenBrowser 差，比 IE 强。（需要插件）
7，支持非常强的保存功能，比 IE 及 GreenBrowser 都强得多。懒虫 MM 的网页总算可以无损保存了。 （需要插件）

缺点，
1。有点兼容性问题。比如清谈发帖，粗体不成功。有些表格也不能成功发送。这是最大的问题。虽然还没那么严重。
2。字体放大倒是很方便，但有时候网页稍乱，缺乏 GreenBrowser 和 Maxthon 那样的整体放大页面的功能。

目前是 kid 的主力浏览器，主要是因为它的功能太强大了，而且可以充分的 DIY ，自己组装。
几乎任何功能都能自己装上去，所以功能强大可以雄踞几大浏览器之首。也非常好玩。

5。Netscape 7.2 （8.0 尚是 BETA 版，不测了。）
界面简单，启动速度不错。
优点：
1。启动速度快的很。
2。安全性好，浏览网页快，
3。支持杀掉广告。但对付漂浮窗口效果不好。
缺点，
1。鼠标拖拽功能没有，
2。下载不方便。
3。杀广告功能还不够完善，这三项都是因为不支持插件而无法弥补。
4。功能还差得不少，比如说整体放大功能，这是唯一没有的浏览器。

多页面作的也不太方便。总的评价和 Opera 差得不多，都是因为
单一公司制作，所以众口难调很难做的尽善尽美。而个人也无法修补什么。

总结，
FireFox 因为极其强大的功能，很好的安全性，非常方便的操作居第一位，略有暇癖是因为和 IE 核心不同。
IE 核心的 GB/Maxthon 因为极好的兼容性，和相当不错的安全性和方便性 居第二位，

Opera 要不是有清谈天地这个论坛的兼容性问题，是可以争一争的，但现在只能望后排，
Netscape 排的更低。这两个落后，主要是因为 FF 太强大了。

IE？苍白的 IE 根本未列在 kid 的视线之内。太多的缺陷使 kid
极少极少启动 IE。
这也保证了 kid 这里从来不中毒不中木马。

XDJM 们如果太习惯于 IE ，kid 也建议用一用 GreenBrowser 和 Maxthon ，肯定会发现很多的惊喜的。

4。Opera ，
过去 kid 的至爱，Opera 8 的启动速度非常快。是最快的。而且界面非常简单明了。功能也很齐全。
优点，
1。启动速度快的很。
2。安全性好，浏览网页快，
3。支持杀掉广告。但对付漂浮窗口效果不好。
缺点
1。兼容性有问题，请谈天地根本没法发帖，----判了死刑。
2。鼠标拖拽功能没有，
3。下载不方便。
4。杀广告功能还不够完善，这三项都是因为不支持插件而无法弥补。

够了，其他的还没有尝试，但最方便的几个功能都不支持。Opera 的评价没法高上去。
