百度前端的七巧板——Tangram JavaScript库
随着百度产品的不断发展和增多，提高前端研发的工作效率成了一个很重要的问题。一个良好设计的、适合自身产品的JavaScript基础库，能让一线工程师从繁重的基础问题中解放出来，将更多的精力用于优化业务逻辑和产品体验，从而提高研发效率。
此次演讲将围绕百度前端JavaScript库Tangram的整体设计思路和实现方式，与大家分享百度在前端方面的经验和心得。
Tangram库是一个开源项目，详见：http://tangram.baidu.com

去哪儿网（Qunar.com）的JavaScript模块化之路
Qunar.com在过去的产品快速发展过程中，也使用了多种优秀的JavaScript开源框架，但因为缺乏合理的控制，导致一些老代码的维护工作相当繁重。为了解决类库之间的兼容性问题，提高代码的可重用性，规范已有的前端代码，并实现自动化测试，Qunar前端团队制作了module.js和qtest。
此次演讲，我会就module.js及qtest在Qunar研发团队中的应用，与大家分享我们在前端方面做的一些努力。
