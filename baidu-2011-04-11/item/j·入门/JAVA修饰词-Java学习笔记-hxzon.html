JAVA修饰词-Java学习笔记-hxzon

abstract
1.抽象类（不能实例化）
2.抽象方法（必须在抽象类中）

static
1.静态成员域，静态成员方法
（属于类，不属于实例。没有实例就可以调用。
占用的是类本身的存储空间，所以任何实例对它进行访问或修改，效果是一样的，即取到相同的值，发生相同的变化。
静态方法不能调用非静态方法和变量，因为它们可能还不存在，在没有实例的时候。）
final
1.最终类（不能派生子类）
2.常量（只能在定义时赋值，或者在构造方法时赋一次值，此后再不能修改）
3.最终方法（不能被子类方法覆盖，即子类不能再声明相同的方法）
================================================
接口interface
1.所有域都是公开，静态，常量（不用显式指明）
2.所有方法都是公开，抽象（不用显式指明）
3.可以继承其它多个接口
4.可用public，abstract，static，strictfp修饰
