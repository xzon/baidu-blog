Abstract Syntax Notation I ,asn.1资源

ASN1 consortium için : http://www.asn1.org

ASN1 hakkinda kitaplar, yazilmis makaleler ve hazirlanmis guzel sunumlar iceriyor.

A Layman’s Guide – Ber/Der farkı için güzel ve geniş bir anlatım :

http://luca.ntop.org/Teaching/Appunti/asn1.html

ASN1 için güzel bir site. Hemen hemen tüm bilgileri içeriyor: http://asn1.elibel.tm.fr/links/

Yukaridaki linkte ASN1 hakkindaki open source calismalarin ve ucretli yazilmlarin cogunun linki mevcut. Ama tum calismalar bu linkte toplanmis diyemeyiz. Olmayanlar var. Ben bu link altindaki ozellikle java kaynakli yazilimlarin hepsine baktim. C/C++ tabanli ASN1 calismalarindan bi iki ornek de inceledim.

Aşağıda kısmen incelediğim ASN1 projelerini anlatmaya çalışacağım. Bunların büyük bir çoğunluğuna bu sitedeki linklerden ulaştım.

com.objsys.asn1j.runtime RtClass paketi

ASN1 api’si. ASN1 encode decode için kullanışlı sınıflar var. Ama bu API’nin .jar dosyasi yok zaten. Object systems’in ASN1 compiler’i ucretli bi yazilim.

Paket yazılımı için fikir vermesi açısından incelenebilir :

http://www.obj-sys.com/docs/acv58/JavaRunTime/index.html

ASN1c : Online asn1 compiler : http://lionet.info/asn1c/asn1c.cgi    

Browse ile asn.1 dosyasi upload edilerek C kaynak dosyalari olusturulabiliyor. Olusturulan (.c ve .h uzantili) kaynak dosyalar(source files) ziplenmis bi sekilde siteden indiriliyor. BER desteği de var.

III ASN : The ASN.1 Tool Open source C++ associated project : http://iiiasn1.sourceforge.net/main.html

C++ dilinde yazilmis open source bir çalışma.

Parser da içeriyor. Kullanım için gerekli işlemler yapıldıktan sonra (build işlemleri yine yukarıdaki linkte anlatılıyor)

asnparser –c temp.asn

şeklinde bi komutla C++ dosyaları ilgili izin(directory) altında oluşturuluyor.

BER encode/decode desteği de var.

Cyrptix ASN.1 Kit DER encoded Java compiler :

http://sourceforge.net/projects/cryptix-asn1/       http://www.cryptix.org/

Der destekli open source bir çalışma.

Dokumantasyonunda yazan şu ifadeler aslında BER encoded compiler bulmanın zorluguna işaret ediyor gibi :

“Cyrptix ASN.1 Kit Future Improvements :

May be add BER encoder and decoder --everybody wants them, but nobody is willing to write them!”

Cryptix in parser kısmında “SableCC” adlı open source bir projeden faydalanılmış:

http://sablecc.org/wiki

SableCC, open source bir parser generator. Dokumantasyonundaki şu ifadeler hakkında genel bir bilgi veriyor :

“SableCC is a parser generator which generates fully featured object-oriented frameworks for building compilers, interpreters and other text parsers. In particular, generated frameworks include intuitive strictly-typed abstract syntax trees and tree walkers. SableCC also keeps a clean separation between machine-generated code and user-written code which leads to a shorter development cycle.”

Better ASN (Java) : http://sourceforge.net/projects/basn
Open source bir çalışma. Java ile yazılmış. DER encode/decode işlemleri yapılabiliyor. Parser kısmı yok.
Readme dosyasında şu şekilde tanımlanmış :
“(developed by Thawte Technologies,) is a Java ASN.1 API that achieves near 100% encoding efficiency for typical ASN.1 usage. An ASN.1 to Java compiler is provided, facilitating custom extension of the API for any application.”
 
 
ARC asn1 to Java compiler :
http://www.forge.com.au/Research/products/arc/arc.htm
Text parser kısmını da içeren Java’da yazılmış open source bir çalışma. JavaCC ve JJTree kullanılmış. 
İlgili yapılandırma(build) işlemleri yapıldıktan sonra
java AsnParser –d source -p <...> temp.asn1
şeklinde bi komut ile  .java  kaynak dosyaları hedef dizinde(directory) oluşturuluyor. Yazılımın .zip’li arşivini indirdiğimizde ReadMe dosyasında BER ve DER hakkında bi bilginin verilmediğini görüyoruz; fakat kaynak dosyaları incelediğimizde sadece Asn1Object.java sınıfının başlangıcında paketin sadece DER destekli oldugunun yazılı olduğunu görüyoruz :
/*
This package supports reading and writing DER encoded
 * ASN.1 structures.
 * BER encoding is not supported.
*/
 
JSS (Mozilla): includes pure Java interface for BER decoding, DER encoding :
http://www.mozilla.org/projects/security/pki/jss/
kaynak dosyalar (source code) :
http://mxr.mozilla.org/mozilla/source/security/jss/org/mozilla/jss/asn1/
projenin API’si :
http://www.mozilla.org/projects/security/pki/jss/javadoc/org/mozilla/jss/asn1/package-summary.html
JSS projesi için birçok open source paket(package) oluşturulmuş. Bunların arasında asn1 paketi de var. Bu pakette BER decoding işlemlerini yapabileceğimiz sınıflar var. Ama encoding Der formatı için yazılmış. BER encoding yapılamıyor.
Ve parser kısmını da içermiyor.
 
JMGMT : BER encode ve decode işlemlerini yapabilen open source bir pakete sahip. Kaynak dosyalar :
http://www.ibr.cs.tu-bs.de/svn/jasmin/trunk/manager/api/java/jmgmt/
JMGMT de parser kısmı yok. Ve ASN1 veri turlerinin hepsi (data types like: integer, sequence, etc.) sadece bi sınıfın < ASN1Value.java > objeleri olacak şekilde düşünülmüş. Bu sınıfta her tur için kullanılacak farklı constructor lar tanımlanmış. Ve Parser kısmını içermiyor.
 
Asn.1 (W. Woody-Caltech): Ber,Der and Cer encoding/decoding library
http://www.chaosinmotion.com/wiki/index.php?title=ASN.1_Library
 
Apache – Asn.1 Framework – (no parser part) and not fully supported BER :
Doc : http://harmony.apache.org/subcomponents/classlibrary/asn1_framework.html
Src : http://svn.apache.org/repos/asf/harmony/enhanced/classlib/trunk/modules/security/src/main/java/common/org/apache/harmony/security/asn1/
