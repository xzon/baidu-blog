JScrollPane不能显示滚动条 -hxzon
   JScrollPane b = new JScrollPane();
final ImagePanel image = new ImagePanel();
b.setViewportView(image);
hxzon：不能显示滚动条，ImagePanel继承自JPanel，覆盖getPreferredSize()方法，解决。
@Override
public Dimension getPreferredSize() {
return new Dimension(1000, 500);
}
