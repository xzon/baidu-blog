MySQL的basedir动态写入 -hxzon
=====================
让MySQL的basedir变为相对路径。
首先，my.ini注释掉basedir这一行。
然后创建脚本，
@echo off
set CURRENT_DIR=%CD%
echo basedir=%CURRENT_DIR%\mysql5>>mysql5\my.ini
将basedir重新写入。示例如下：
-------------------------
@echo off
set CURRENT_DIR=%CD%
echo basedir=%CURRENT_DIR%\mysql5>>mysql5\my.ini
start /min mysql5\bin\mysqld-nt.exe --defaults-file=mysql5\my.ini
ping -n 5 127.0.0.1>nul
mysql5\bin\mysql.exe -u root -proot helloDatabase < helloDatabase.sql
ping -n 3 127.0.0.1>nul
mysql5\bin\mysqladmin --user=root --password=root shutdown
----------------------
echo [message]>[file name] 这个命令是把指定的字符串送到指定的文件中
注意我的说明:
“>>”将把内容将填加到文件中，“>”将原文件内容覆盖
