Can not find a java.io.InputStream with the name [targetFile] in the invocation stack. -hxzon动手 -gdo

struts2下载文件出错。

严重: Servlet.service() for servlet default threw exception
java.lang.IllegalArgumentException: Can not find a java.io.InputStream with the name [targetFile] in the invocation stack. Check the <param name="inputName"> tag specified for this action.

实际问题是ServletActionContext.getServletContext().getResourceAsStream(path);为null，
原因是path是文件名（即相对应用名之后的路径），而我写成全路径。
当把path写成"upload/1.txt"时成功。
