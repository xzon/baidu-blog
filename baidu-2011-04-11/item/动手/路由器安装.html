路由器安装
时间：2006-6-9 作者：网络安装

硬件安装

1. 断开所有设备的电源，包括计算机、调制解调器和路由器。

2. 用网线连接局域网中的计算机网卡端口到路由器后面板LAN端口。（如果使用Uplink，则相应复用端口将失效；如果是BEFSR11,则要将路由器接到集线器或交换机上）

3. 调制解调器与路由器后面的WAN端口相连。

4. 给路由器后面的电源端口接上电源电缆，然后把提供AC电源的电缆插入电源线，电源线的另一端插入电源插座。

a) 电源接通时，电源灯（Power）亮（绿色）

b) 当路由器进行内部诊断测试时，Diag灯持续几秒钟呈红色。自测完成后灯灭。

5. 接通调制解调器电源。

6. 按压路由器前面的Reset按钮，并保持3秒钟或到Diag灯呈红色。这将恢复路由器的默认设定。

配置您的路由器

1. 打开您的浏览器在地址栏中键入http://192.168.1.1。这个地址就是您的路由器IP地址。按回车。

2. 用户名和口令对话框将出现。保留用户名栏空，口令栏中键入admin（默认的口令）。点击“确定”。

注∶如果您以前已在您的局域网上使用过代理服务器，现在您必须禁止它。

如果您正在运行网景浏览器，点击Edit（编辑）＞＞Preference（优先）＞＞Advanced （高级的）＞＞Proxies 和点击“Direct Connection to the Internet(直接连接到Internet)”

如果您正在运行IE5或更高版本，点击开始（Start）＞＞设定（Setting）＞＞控制台（Control panel） ＞＞Internet选项（Internet options）＞＞联机（Connections）＞＞局域网络设定（LAN Settings）。勾选“自动侦测设定（Automatically Detect Settings）”。点击“确定”继续。

3. 路由器设置页面出现，配置下列参数

Router Name & Domain Name(路由器名和网域名)

这两个参数允许您为路由器提供主机名和域名解析。一些ISP需要这些名称作为标识。您必须咨询您的ISP，看您的宽频国际互联网服务是否已经配置了主机名和网域名。在大多数情况下，保留这两个参数为空也能正常工作。

LAN IP Address (局域网 IP地址)

这些值涉及您的内部局域网设置。除非您有特殊内部需要，就不应当改变这些值。对于内部LAN其默认值如下述：

专用IP地址：192.168.1.1

子网掩码（Subnet Mask）：255.255.255.0

WAN IP Address(广域网 IP 地址)

这些参数用在您每次访问互联网时。大多数宽频ISP在客户每次登陆时指定客户机不同的IP地址。若是您的ISP是这种情况，点击Obtain an IP Address Automatically (自动获得IP地址)并继续到步骤5。若您的ISP指定您固定IP地址，点击Specify an IP Address(指定IP地址)并输入地址、子网掩码、默认网关和DNS。

4. 当您已适当地配置了设置页，点击Apply（应用），然后点击Continue（继续）。

5. 在网页上点击DHCP标签

6. 除非您的内部网络上已有一个DHCP服务器，应选择Enable。选Enable,您的路由器将自动分配IP地址给您的局域网计算机。在Number of DHCP Users（DHCP用户数）框中，输入您打算由路由器分配IP地址的计算机数。将来您增加计算机的时候不要忘记变更此数值。

7. 点击Apply（应用），然后点击Continue（继续）。

8. 重新启动调制解调器和您的PC机以使得PC获得新的路由器配置信息。

配置您的局域网连接到路由器

1.点击开始（Start）按钮，选取设定（Setting）>>控制台（Control Panel）。

2.双击网络（Network）图标。

3.在“配置(Configuration) ”窗口选择已与您的网络卡/适配器相联系的TCP/IP协议。

4.点击“内容(Properties)”，选择“自动获取IP地址(Obtain an IP address automatically)”。按压“确定”。您已完成了用户设定。

5. 点击“确定”。Windows可能要求原始Windows安装文件，按其需要提供(如：D:\Win98, D:\Win95, C:\Windows\Option\Cabs )。

6．窗口将要求您重新启动计算机。点击“确定”。

对您局域网的每一台计算机重复1-6步骤。当您的所有计算机配置好后，路由器的安装和设置就完成了。
