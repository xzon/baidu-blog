windows xp安装PHP ;安装joomla
hxzon:
1,下载运行
httpd-2.2.15-win32-x86-openssl-0.9.8m-r2.msi
2,下载运行
php-5.2.13-win32-installer.msi
中途会选择Apache2.2的conf路径。
3，放置php网站到Apache2.2的htdocs路径下，并不能运行。
进入conf\httpd.conf文件，
找到239行，添加index.php，即改为
<IfModule dir_module>
DirectoryIndex index.html index.php
</IfModule>
到达文件末端，php安装程序已写入
#BEGIN PHP INSTALLER EDITS - REMOVE ONLY ON UNINSTALL
PHPIniDir "D:/develop/PHP/"
LoadModule php5_module "D:/develop/PHP/php5apache2_2.dll"
#END PHP INSTALLER EDITS - REMOVE ONLY ON UNINSTALL
但还需要写入
AddType application/x-httpd-php .php
重启Apache，可以运行php网页。
-----------------
4，安装开源内容管理系统joomla1.5.15
将joomla1.5.15目录放入htdocs路径下，
浏览器http://localhost/joomla1.5.15/index.php
进入joomla1.5.15安装界面。
注意需要先创建数据库。
create Joomla;
GRANT ALL PRIVILEGES ON Joomla.*
TO nobody@localhost IDENTIFIED BY 'password';
安装完成后删除htdocs\joomla1.5.15\installation目录。
开始使用网站。
http://www.mysite.com -> Main Site
http://www.mysite.com/administrator -> Admin
