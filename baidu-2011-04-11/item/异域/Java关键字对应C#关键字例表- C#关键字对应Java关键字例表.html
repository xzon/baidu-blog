Java C#    意义

class Class 类
interface interface 接口
extends :      继承类
implements     :          实现接口
public     public           公有的
protected      protected 只继承有用
private       private     私有的
abstract     abstract        抽像的
final        sealed    不可在继承的
super               base                 调用父类的方法

Java C# 意义

package namespace 用來將類別分組的方法,
     但package既要宣告, 也要用檔案目錄來包裝, namespace只要宣告即可
import using     用來將上述的類別組的宣告引進目前的程式碼中
.jar .exe/.dll    用來儲存許多類別的檔案
ClassLoader         Assembly     用來載入上述檔案的類別
(不用寫) internal     限制使用的, Java是在同一個Package中可使用, 但C#是在同一個.exe/.dll中可使用
