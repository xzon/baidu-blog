一，小费马定理

a^(p-1)=1 mod p,其中gcd(a,p)=1.

证明：首先，明确p不一定是素数，不要求a<p.

可能的错误1，a<p.

可能的错误2，a,p互质，以为a mod p=1.与最大公约数混淆。

导致以为(p-1)a mod p= ((p-1)mod p)* (a mod p)= (p-1)*1= p-1.

先证明，当p为素数，那么对任意一个与p互质的正整数a，

a, 2a ,到(p-1)a除以p的余数正好是1到p-1的集合。

反证法。假设不成立，那么则有两个数m，n(m,n<p),

使得ma mod p=na mod p,那么(m-n)a mod p=0.

这是不可能的，因为(m-n)小于p，

同时a与p互质，都不可能含因子p。

因此，有(a mod p)( 2a mod p)……((p-1)a mod p)= 1*2*……(p-1),

[a* 2a *……(p-1)a] mod p= 1*2*……(p-1),

(a^(p-1))*(p-1)! mod p= (p-1)!,

a^(p-1)=1.

二，欧拉函数

φ(n)=(p-1)(q-1),其中p，q为素数，n=pq。

证明：构造集合{1,2,3,……pq},其中与n不互质的元素为

{p,2p,3p,~~(q-1)p}及{q,2q,3q,~~(p-1)q}及0,

所以φ(n)=pq-(q-1)-(p-1)-1

= pq- (p+q)+1

=(p-1)(q-1).

二，欧拉定理

a^φ(n)= 1 mod n

证明：可能的错误1，以为n是素数，所以φ(n)=n-1,依据小费马定理，得到。

实际上，n是任意整数。

注意φ(n)是指小于n且与n互质的正整数的个数。

考虑这些整数的集合：

R={x1,x2,……xφ(n)},

用a与R中的每个元素相乘mod n,得到下述集合

S={ (ax1 mod n), (ax2 mod n),……(axφ(n) mod n) },

集合S是R的一个置换，原因如下：

1，a与n互质，且xi与n互质，所以axi必与n互质。这样S中的所有元素均小于n，且与n互质。

2，S中没有重复元素。因为若axi mod n=axj mod n，则xi=xj.

(a与n互质，若(a*b)=(a*c)mod n,定理有b=c mod n).)

因此我们有(ax1 mod n) (ax2 mod n)……(axφ(n) mod n)= x1 *x2 ……*xφ(n),

ax1 *ax2……*axφ(n)= [x1 *x2 ……*xφ(n)] mod n,

aφ(n)* [x1 *x2 ……*xφ(n)]= [x1 *x2 ……*xφ(n)] mod n,

aφ(n) =1 mod n.

特别的，当n是素数时，φ(n)=n-1,即小费马定理。

三，欧拉定理另一种有用的表达形式

a^[φ(n)+1]= a mod n.

四，欧拉的一个推论

给定两个素数，整数n=pq，和m，其中0<m<n,

则m^[φ(n)+1]= m^[(p-1)(q-1)+1]= m mod n.

五，RSA原理

已知，ed=1 mod n,m<n.(注意这里的m<n)

加密c= m^e mod n,m= c^d mod n.

证明：c^d mod n

= (m^e mod n)^d mod n

= m^ed mod n

= m^(kφ(n)+1) mod n

= [m^(kφ(n)) mod n] [m mod n]

= m mod n(因为m<n)

= m

六，连续两个整数互质

证明：辗转相除法，gcd(n,n-1)=gcd(n-1,n mod n-1)=gcd(n-1,1)=1,得证。

七，若n为偶数，则φ(n)为偶数

证明：已知n=2k,k>1.

那么{1,2,3……n-1}与n互质的元素个数，和

{1,3,5,……n-1)与n互质的元素个数相同。

对任意一个奇数x，gcd(x,n)=gcd(x,k).

(因为x为奇数，所以最大公约数不可能是2，从而约去)。

{1,3,5……n-1}与n互质的个数，

等于{1,3,5……k-1}与k互质的元素个数+0+{k+1,k+3,k+5……2k-1}与k互质的元素个数。

记为x+0+y.

因为(k+i)mod k= i mod k,所以x=y.

所以φ(n)=2x，为偶数。
