body
<p>段落文字</p>
<br>换行
<h1..6>正文标题</h1..6>(数字表示文字大小)
注释标记: <!--注释文字-->
水平分隔线标记:<hr>   注意属性参数效果


head
文档头部标记，主要供浏览器或搜索引擎使用，而不会显示在网页正文中。
主要子标记（子元素）：
<title>-设置窗口标题 如：    <title>工作室图书信息</title>
<link>-建立到外部文件（主要是CSS外部样式表）的连接 如： <link rel="stylesheet" herf="mystyle01.css" type="text/css">
<style>-设置网页的内部样式表 如：   <style type="text/css"> body{background-color:white;color:red;}</style>
<meta>-设置当前页面的元数据信息 如：(主要用于介绍当前页面信息，方便搜索引擎显示)
   <meta name="description" content="HTML实用教程">
   <meta name="keywords" content="HTML,v512工作室，教程">
   <meta name="author" content="zlg">
   <meta http-equiv="Content-Type" content="text/html;charset=GBK"> （编码方式）
   <meta http-equiv="refresh" content="5;url=http://www.v512.com"> （定时刷新）

     
文本格式（字体格式）
<b>粗体字</b>    <i>斜体字</i>   <del>词当中划线表示删除</del> <ins>词下划线表示插入/填充内容</ins>
下标:<sub>1</sub> 上标:<sup>7</sup>

  
<pre>
预设(preformatted)文本，原样显示...
          保留    文本中的空      格和

   换行。
</pre>


字体
<font> 标记用于设置字体的类型、大小和颜色。
<font>标记的属性:
    face- 设置字体类型 如：    <font face="宋体">文字内容</font>
    size- 设置字体大小 如：    <font size="2">文字内容</font>
    color -设置字体颜色 如:    <font color="#FF0000">文字内容</font>

图片 <img>标记用于HTML页面中插入图片
<img src="young.png">   相对路径
<img src="http://www.v512.com/attachments/book.gif">   绝对路径
<img>标记的其他属性
alt-在不支持图片显示的浏览器中将显示本属性值(alternate text) 如: <img src="young.png" alt="悠悠的照片">
width/height- 设置图片的大小
align- 设置图片的水平和垂直对齐方式
border -设置图片边框线条宽度


特殊字符显示

超链接 <a href="http://www.baidu.com">
<a> 标记的其他属性：
    target-在指定的窗口或新窗口中显示连接页面 如：<a href="http://www.google.cn" target=_blank>google</a>
    name- 设置锚(anchor)标记,实现超链接转到页面指定位置(实现跳转到同一页面不同部位) 如：
                          <a name="p1">文本内容</a>
                           <a href="#p1">跳转到（锚标记p1）指定的位置</a>
    title- 设置超链接的说明文字（鼠标悬停在超链接上时显示）
             <a href="young.png" title="点击可以看到照片">7仔</a>
    连接到email地址：<a href="mailto:cwy-no.1@163.com">联系我</a>
    图片作为超链接 <a href="..."> <img src="young.png"> </a>

有序列表(order list)
<ol type="a">
<li>列表条目1</li>
<li>列表条目2</li>
</ol>

无序列表(unorder list)
<ul type="disk">
<li>列表条目1</li>
<li>列表条目2</li>
</ul>

定义列表(defintion list)
<dl>
<dt>列表条目1标题</dt><dd>列表条目1正文</dd>
<dt>列表条目2标题</dt><dd>列表条目2正文</dd>
</dl>


表格的HTML相关标记(<table><tr><td><th><caption>)
<table>标记一章表格
<tr>标记表格中的一行
<td>标记表格中一行中的一列或一个单元格
<th>用来标记表头中第一行中的单元格
<caption>表格的标题
表格的通用相关标记属性：
width/height-- 指定表格或某一个单元格的宽度、高度
border-- 指定边框线条的宽度
bordercolor--边框颜色
bgcolor-- 指定表格或某一单元格的背景颜色
background-- 指定表格或某一单元格的背景图片
align-- 设置单元格对齐方式
cellspacing--设置单元格间距
cellpadding-- 设置单元格内容与单元格边界之间的距离（类似合并单元格）
colspan/rowspan-- 实现跨行/跨列单元格 如：
<td colspan="2">xxx</td> 跨了2列的单元格


表单（重点）
HTML表单用于收集和提交用户输入的信息
常用的信息输入形式：单行文本、多行文本、单选按钮、多选框等、选项式列表、密码和文件
基本语法格式：
<form action="http://www.google.cn" method="post">
用户名：<input type="text" name="username"><br>
密码: <input type="password" name="psw"><br>
<input type="submit" value="提交">
<input type="reset" value="重置">
</form>
HTML表单的两种提交方式: method="get"   可以看到表单参数，以问号为分隔，优点速度快，缺点：不保密、容量小
                        method="post" 可以看见提交页面的url,但看不到提交的信息
表单组件
单行文本输入框   <input type="text" name="age" value="0">

密码输入框       <input type="password" name="psw">

单选按钮 <input type="radio" name="gender" value="male" checked>男
                 <input type="radio" name="gender" value="female">女

复选按钮         <input type="checkbox" name="fruit" value="apple">苹果<br>
                 <input type="checkbox" name="fruit" value="grap">葡萄<br>
                 <input type="checkbox" name="fruit" value="orange">桔子<br>

下拉列表框        <select name="fruit" size="3" multiple>
                     <option value="apple">苹果
                     <option value="grap">葡萄
                     <option value="orange">桔子
                   </select>

多行文本输入框    <textarea name="info" cols="40" rows="5">请留言:</textarea>

提交按钮          <input type="submit" name="submit" value="提交">

重置按钮          <input type="reset" value="重置">

图片提交按钮   <input type="image" src="dong.gif" alt="提交" name="submit2">

隐藏文本框        <input type="hidden" name="role" value="guest"> (可以配合session cookies使用)

文件组件   <form action="k.jsp" enctype="mutipart/form-data" method="post"> (上传文件，注意设置enctpe / post)
                  <input type="file" name="f1">
                  </form>


页面框架（使用一个浏览器将窗口分割成多个窗格，以显示多个不同的页面）
行分割：
     <html>
          <frameset rows="25%,">
            <frame src="a.html">
            <frame src="b.html">
           </frameset>
     </html>
列分割:
      <html>
           <frameset cols="300,400,*" border="0">
              <frame name="myframe1" noresize src="a.html"> (noresize分割后不可变化大小)
              <frame name="myframe2" src="b.html">
              <frame name="myframe3" src="c.html">
            </frameset>
       </html>
混合分割和页面导航


嵌入标记
<applet>标记用于在页面中嵌入JAVA applet
<embed>标记用在页面中嵌入多媒体文件，但用户需在计算机 上需事先安装好相应的处理程序
常用嵌入文档格式：mp3/mid/wma/asf/rm/ra/swf/avi
标记用法:
<embed src="v512.avi" autostart="true" loop="true" hidden="false" controls="console" width="200" heighy="45">
