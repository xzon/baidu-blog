9，String
String s1="abc";
String s2="abc";
String s3=new String("abc");
String s4=s3. intern();
s1==s2为true，因为使用字符串常量声明，放在了字符串池。hxzon验证。
s1==s3为false，因为生成了新的字符串对象。hxzon验证。
s1==s4为true，因为intern()方法查找字符串池，如果存在该字符串常量则返回字符串池中该字符串的引用，否则在字符串池中新生成一个。hxzon验证。

应用了Flyweight模式。

10，2.0默认为double类型，2.0f为float类型。
public static void a(Float a){
   a=a*2;
}
Float test = 2.0f;
   Final.a(test);
   System.out.println(test);
输出是2.0而不是4.
当同时定义a(Float a)和a(float a)时不会出现歧义。而只定义其中一种时，会自动装箱拆箱。

11，for先判断条件再执行。而不是先执行一次再判断。
int i=3,j=0;
   for(--i;i>=0;){
    j++;
    System.out.println(i+","+j);
   }
   System.out.println("end");
会不断循环。因为--i只执行一次，容易忽视。
