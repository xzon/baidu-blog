Firefox 3: 你可能不知道的8件事（设置标签上的关闭按钮）

http://pic.yupoo.com/jdvip/783786451834/small.jpg


相比于可能已经被你遗忘的 Firefox 2 来说，Firefox 3 在性能上有大幅度的提升。你可能感觉到它的强大，但下面这些小细节或许没有注意到。

1、拖放即可复制标签
当然，你得同时按住 ctrl。这个对于某些不能新开窗口的 js 按钮很有用。

2、最小化工具条
在工具条上右键，选择“定制”，然后勾选“使用小图标”。(其实这个早就有了，作者火星不关我事 - -)

3、智能书签

http://pic.yupoo.com/jdvip/6287364517d5/medium.jpg


Firefox 3 的用户肯定注意到书签栏第一个按钮能显示你访问频率最高的最近10个链接，它是怎么实现的呢？其实你可以定义自己需要的。比如自带的那个，其实新建一个名为“最近访问”地址为“place:queryType=1&sort=12&maxResults=10”的书签就可以了。这里有一份很长的官方命令列表，根据它你可以 diy 出很强大的专有个人书签。

4、将 yahoo! 邮箱 或 Gmail 作为默认邮件客户端

http://pic.yupoo.com/jdvip/1835464517d7/medium.jpg


以前点击邮箱链接一般是挑出讨厌的 outlook 的，现在不用复制 email 地址再进入 Gmaill 了。我们可以在设定好 gmail 作为默认邮件发送端：点击菜单的“工具 ->选项 ->应用程序”，然后搜索 mailto，将默认的“总是询问”改成“使用Gmail”，确定。如果你的 Firefox 版本里没有“使用Gmail”选项，可以打开 about:config 页面，在过滤器输入 gecko.handlerService.AllowRegisterFromDifferentHost，然后双击它的值改为 true。然后在地址栏输入
javascript:window.navigator.registerProtocolHandler("mailto", "https://mail.google.com/mail/?extsrc=mailto&url=%s","Gmail")
然后可以看到Gmail的添加提示了(但是我点击我们的 联系信箱 链接却没有测试成功 - -)。

5、设置标签上的关闭按钮

http://pic.yupoo.com/jdvip/3606964517d8/medium.jpg


Firefox 3 默认是在每个标签上都有关闭按钮的，你可以设置他们是否可见。打开 about:config，在过滤器中输入 browser.tabs.closeButtons，然后选择下面几个值之一：
0 / 仅在当前标签显示关闭按钮
1 / (默认)在每个标签显示关闭按钮
2 / 所有标签都不显示关闭按钮
3 / 仅在最右端显示关闭按钮以关闭当前标签

6、设定 Awesome Bar
Firefox 3 的地址栏因为功能更复杂而增加 Awesome Bar，它能根据具体情况而显示你需要的设定。默认它会记录你的浏览历史，你可以打开 about:config，然后进行详细设定：
/ 完全禁止 Awesome Bar 功能而返回 Firefox 时代，可将 browser.urlbar.maxRichResults 的值改为 -1
/ 只允许 Awesome Bar 记录那些你直接输入地址打开的网站，可将 browser.urlbar.matchonlytyped 的值改为 true
/ 移除推荐内容里你从未访问的站点，可将 places.frecency.unvisitedBookmarkBonus 的值改为 0
/ 从 Awesome Bar 里移除所有书签，请将 places.frecency.unvisitedBookmarkBonus 和 places.frecency.bookmarkVisitBonus 的值都改为 0
