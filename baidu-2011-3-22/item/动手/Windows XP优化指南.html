Windows XP安全隐患

 首先提醒大家，微软Windows XP的通用即插即用（Universal Plug and Play — Upnp）属性中有个重大的安全缺陷。这个安全缺陷使攻击者可以向计算机发送一个伪造的“缓冲溢出”包，从而获取对系统的完全控制。微软很快发布了相应的补丁，请所有的XP用户下载并运行一下。另外，装有XP的Internet连接共享客户端的Windows 98或ME也会受到这种的攻击，微软还发布了这些操作系统的相应补丁。请大家下载并安装相应的补丁程序安装，以确保系统安全：

Microsoft XP (Home & Pro Edition) - 通用即插即用补丁 —— 点击这里下载

Microsoft Millenium Edition - 通用即插即用补丁 —— 点击这里下载

Microsoft 98/98SE - 通用即插即用补丁 —— 点击这里下载

想要了解相关补丁或安全漏洞的技术细节，请参阅微软官方公告。

下面我们从简单细微的优化开始讲起，越到后面的优化越有效、精彩，不要看走喽……

优化1——共享文档

http://www.southcn.com/it/itschool/200201101060_41961.gif



默认情况下，在Windows XP中打开我的电脑，会看到在硬盘图标上方有一些文件夹。这些就是“共享文件夹”（看上图粉红色区域），这里有每一个用来户共享文件所用的文件夹。个人认为，这些文件夹特别烦人，毫无必要。我们可以让这些文件夹在我的电脑中消失，原理很简单，只要打开注册表找到如下位置：HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\
Explorer\MyComputer\NameSpace\DelegateFolders，把{59031a47-3f72-44a7-89c5-5595fe6b30ee}键值删掉，下次打开我的电脑，这些烦人的文件夹就不复存在了。

优化2——清除还原点，释放硬盘空间

Windows XP有创建“还原点”的功能，存储包含系统设置信息的简单文件，以防系统崩溃时，设置丢失。然而，一定情况下Windows会创建多个不必要的还原点，有时会占用几G的硬盘空间。我们要把除最近还原点之外的其它文件删掉，只需点击开始—>程序—>附件—>系统工具—>磁盘清理。

http://www.southcn.com/it/itschool/200201101060_41963.gif



选中要清理的硬盘，然后点“清理...”，最后点击确定，就完成了，再看看硬盘剩余空间，可能已经多了几G！

优化3——快速关机

修改几个注册表键值，就可以大大减少Windows关闭所用的时间。首先打开注册表编辑器，找到HKEY_CURRENT_USER\Control Panel\Desktop\，里面有个名为HungAppTimeout的键，它的值应该是5000（默认值，如果不是，把它改为5000）。接下来，还有个WaitToKillAppTimeout键，把它的值改为4000（默认值是2000）。

http://www.southcn.com/it/itschool/200201101060_41965.gif



最后，找到注册表如下位置：

HKEY_LOCAL_MACHINE\System\CurrentControlSet\Control\

同样地，把其中的WaitToKillServiceTimeout键值改为4000。另外，把控制面版 -> 管理工具 -> 服务中的NVidia Driver Help服务设为手动，也可以加快Windows关闭时间。

优化4——设置IDE设备的DMA模式

 出于某些原因，Windows XP有时会在IDE通道上使用PIO并行输入输出传输模式，而不是DMA模式。如果有这种情况，用户可以手动把IDE通道改为DMA模式，这样可以减少该设备占用的CPU周期。打开设备管理器（右键点击我的电脑，选属性，点选硬件选项卡，点击“设备管理器”打开），然后点击展开“IDE ATA/ATAPI 控制器”分支，双击“Primary IDE Channel”，点击高级设置。

http://www.southcn.com/it/itschool/200201101060_41967.gif



检查“传输模式”下拉菜单，把它们全部设为DMA If Available（允许情况下使用DMA），点击确定。再用同样的方式修改每一个IDE控制器。

优化5——几个重要的网络修正

严格地说，这不算是优化，而是一种错误修正。笔者第一次安装XP就碰到这样的问题，网络与其它用户间不能连接，就连Guest帐号也不能登录，别的用户连不上自己的机子，也打不开共享文件夹，还有一系列的网络怪问题。要修复这个网络问题，请打开控制面版，到管理工具，双击打开本地安全策略。接下来打开本地策略—>用户权限指派。

http://www.southcn.com/it/itschool/200201101060_41969.gif



找到名为“从网络访问此计算机”的设置项，双击它，确保“Everyone”在列表之中。

http://www.southcn.com/it/itschool/200201101060_41971.gif



如果在接受别机从网络访问时有困难的话，在这样加上“Everyone”应该可以解决问题。

优化6——XP Prefetcher

 Windows XP自带了一个名为Prefetcher的服务，这个服务管理着Windows启动时的程序初始，其中指定的程序可以在以后使用中快速载入。Prefetcher在默认情况下就是打开的，不过它的性能还可以进一步提升。打开注册表编辑器，在注册表中找到：

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory
Management\PrefetchParameters，其中有一个键值名为EnablePrefetcher，多数情况下它的值是3。推荐设置值是5——在我的机子上，设为5的时候工作状态最佳。可以随便试试不同的数值，直到找到最适合自己机子的值为止。

http://www.southcn.com/it/itschool/200201101060_41973.gif



也可以把Prefetcher禁用掉，只需把值设为0就可以了。不过，除非是为了测试用途，一般人都不会傻到把它设为0。

优化7——压缩文件夹

这是一个相当好的优化，Windows XP内置了对.ZIP文件的支持，我们可以把zip文件当成文件夹浏览。不过，系统要使用部分资源来实现这一功能，因此禁用这一功能可以提升系统性能。实现方法非常简单，只需取消zipfldr.dll的注册就可以了，点击开始—>运行，敲入：

regsvr32 /u zipfldr.dll

然后回车即可。

优化8——不加载DLL文件

http://www.southcn.com/it/itschool/200201101060_41975.gif



浏览器在使用之后往往会有部分DLL文件在内存中缓存好长一段时间，相当大一部分内存会被不用的DLL文件点用。为了防止这种事情发生，找到注册表如下位置：

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer

接下来建立一个名为AlwaysUnloadDLL，值为1的双字节值。

http://www.southcn.com/it/itschool/200201101060_41977.gif



如果想要去除这项优化，只需把该键值设为0（也可以干脆把键删掉）。注意：该项优化要在Windows重新启动后才生效。

优化9——禁用休眠，释放硬盘空间

 另一个简便的优化——如果不准备使用休眠模式（我数用户都不用的），确保把它禁用掉，因为Windows为实现睡眠模式，在硬盘上预留了跟本机物理内存一样大小的空间，禁用睡眠模式，硬盘的这部分空间就会多出来。要禁用休眠功能，打开控制面版—>电源选项—>休眠选项卡。

http://www.southcn.com/it/itschool/200201101060_41979.gif



正如大家所见，对于内存特别多的用户来说，禁用休眠可以释放出容量相当可观的硬盘空间。

优化10——带宽优化

Windows XP新增了QoSRRP（这是Quality of Service Resource Reservation Protocol的缩写，意为服务质量资源预留协议）。一般来说，试图通过可用带宽访问信息的时候，应用程序要么通过QoS应用程序接口，要么通过另一个称为TCI的应用程序接口。QoS与TCI之间的差别是，我们可以为QoS应用程序预留带宽。这对于QoS应用程序来说无疑是件好事，但如果根本就不用这种接口的应用程序，无形中就有部分（默认为20%）带宽白白浪费了。

要改变这一设置，请点取开始—>运行，然后敲入gpedit.msc，在出来的窗口中打开如下分支：

计算机配置 -> 管理模版 -> 网络 -> QoS包列表。

http://www.southcn.com/it/itschool/200201101060_41981.gif



双击窗口右边的“限制预留带宽”设置，把它设为“可用”，并把带宽限制设为0%。

http://www.southcn.com/it/itschool/200201101060_41983.gif



这样，带宽就比原来增加了20%！

优化11——优化二级缓存

 L2，也就是二级缓存，是CPU的一个组成部分。然而，XP对它的检测非常蹩脚，最好还是手动设定。做法非常简单，只要找到注册表如下位置：

HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory
Management\

找到名为SecondLevelDataCache的键值，把它的值（以千字节为单位）调整为所用CPU的二级缓存实际大小。

http://www.southcn.com/it/itschool/200201101060_41985.gif



CPU二级缓存的大小值可以从厂商处获知。

注：可选用十六进制或十进制填入大小值，应该点选十进制。
