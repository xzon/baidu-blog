为丑陋的摆渡空间添加美观的代码框线
继续改造改造丑陋的摆渡空间。
首先要添加代码框线。
摆渡的空间显然不适合IT写作。因为它并没有定义代码区域的特殊风格，这导致代码一旦贴在空间里，就会和正文夹杂在一起，立刻失去了可读性。
我想了个hack的方法，虽然用起来有点麻烦，但看看这篇包含代码的文章，效果还是不错的。
方法:
在css文件中搜索"cnt"，找到下面这行(或者有类似的两行)
#m_blog div.cnt{color:#333333;line-height:20px;font-size:14px}
            把找到的部分替换成
            #m_blog div.cnt{padding:10px;font:11px/18px "Verdana";}
#m_blog div.cnt pre{font:11px/18px "Verdana";}
#m_blog div.cnt pre pre{margin-top:5px;margin-bottom:5px;padding:5px;background:#EBEBEB;border:1px dotted gray;font:12px "Courier New";}
            以后如果要发表包含代码的文章，还要多一个步骤。
            比如某篇文章内容是:
            这是一篇文章。代码片段一:
            int a = 1;
            int b = 2;
            int c = a + b;
            代码片段二:
            if(condition){
               do something;
            }else{
            ...
            }
            要做以下几个修改。
            1.在一款纯文本编辑器(比如记事本)中编辑你的文章，做完换行，空格等格式布局。
            修改文章，将代码部分用<pre></pre>标签括起来。然后把文章整体再用一个<pre></pre>标签括起来。
            修改后的原文应该是
            <pre>
            这是一篇文章。
            代码片段一:
            <pre>
            int a = 1;
            int b = 2;
            int c = a + b;
            </pre>
            代码片段二:
            <pre>
            if(condition){
               do something;
            }else{
            ...
            }
            </pre>
            </pre>
            2.保存此文件为xxx.html，双击使用浏览器打开。
            3.ctrl+A全选，ctrl+c复制，ctrl+v粘贴到“发表新文章”的内容框里。
            4.如需要调整除代码外的其他部分的字的颜色等，调整之。
            5.发布。
            代码框线添加好了，顺便说说给文章中出现的图片增加虚线边框，使得文章和图片的结合看起来更自然。查看效果。
            方法:
            在css中添加一个标签选择器:
            img.blogimg{margin-left:20px;margin-top:10px;margin-bottom:10px;border:1px dotted black;}
