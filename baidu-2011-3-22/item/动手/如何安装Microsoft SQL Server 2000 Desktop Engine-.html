如何安装Microsoft SQL Server 2000 Desktop Engine?
2007-03-25 02:41
   SQL Server 2000数据安装分为两种：一是SQL Server 2000原版数据库完整安装，二是MSDE（Microsoft SQL Server 2000 Desktop Engine）安装。SQL Serevr 2000完整安装，请按照SQL Server 2000的安装向导进行安装即可，下面主要介绍一下MSDE的安装方法:
  • 第一步：取得MSDE
    • 如果你没有MSDE安装程序，可以通过以下地址免费下载：http://www.microsoft.com/downloads/details.aspx?FamilyID=413744d1-a0bc-479f-bafa-e4b278eb9147&DisplayLang=zh-cn ,下载完毕后，双击打开CHS_MSDE2000A.EXE文件，将其解压到C:\MSDERelA目录。
  • 第二步：安装MSDE
    • 安装MSDE请不要直接打开C:\MSDERelA\Setup.exe进行安装，而是用DOS命令进行安装，具体方法如下：
    • （1）进入DOS   如果是Win 2000、WinXP，选择“开始-->程序-->附件-->命令提示符”；
      如果是Win98，选择“开始-->程序-->MS-DOS提示符”
    • （2）执行MSDE安装命令 命令格式如下：
    • C:\MSDERelA\setup sapwd="abc" securitymode=SQL disablenetworkprotocols=0
      • 注意：
        • A、上面命令行中的C:\MSDERelA 是MSDE的解压文件所在目录，如果解压目录不是C:\MSDERelA请换成相应目录即可；
        • B、"abc"是数据库超级用户sa的口令，此处仅以设成"abc"举例，您可以随意设置，但一定要记住，因为在后面安装配置软件中要用到。
  • 第三步：启动SQL服务
    • 软件安装完毕后，如果SQL服务没有自动启动，请手动启动它，方法如下：打开“我的电脑”，找到文件“C:\Program Files\Microsoft SQL Server\80\Tools\Binn\sqlmangr.exe”（假设您把MSDE安装在C盘上，如果安装在D盘上就要到D盘的Program Files目录下寻找），双击运行。出现类似以下的窗口。

    • 单击“开始/继续”按钮，当左下角图标的圆圈显示为向右绿色三角符时，说明SQL服务启动成功。
    • 选中“当启动OS时自动启动服务（A）”复选框，则每次启动Windows时，SQL就会自动启动。
    • 当SQL服务启动成功后，Windows任务栏的右下方会显示一个图标

      http://www.edingsoft.com.cn/softimage/sql2.jpg

      。

      至此，MSDE就安装成功了。
