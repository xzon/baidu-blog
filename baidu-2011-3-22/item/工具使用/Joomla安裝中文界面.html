Joomla安裝中文界面

hxzon:ok

完成Joomla安裝後，預設語言是英文 (en-GB)，網站開發人員按照客戶的需求，可以為 Joomla安裝中文語言界面，甚至可以使用 JoomFish 把 Joomla 設定成多語言的網站 (Multilingual) 。

http://www.hksilicon.com/kb/content_images/1/vince/zh-admin-default.gif

Pic1: 後台預設是英文界面


語言界面分為前台 (Frontend) 和後台 (Backend / Admin).
前台是一般用戶看到的界面，後台是需要有 Admin 權限才能進入的界面:

  • 前台: http://www.yourwebsite.com
  • 後台: http://www.yourwebsite.com/administrator


你可以選擇只為前台安裝中文版，或者前台後台同時安裝:

Joomla 中文化的步驟

1) 取得中文語言包:

到: http://www.joomla.org.tw 下載最新版本的繁體中文語言包，前後台的語言包是分開的:

  • 前台: zh-TW.site.1.5.xx.zip
  • 後台: zh-TW.administrator.1.5.xx.zip

2) 安裝

2.1 - 登入後台 > Extensions > Install

http://www.hksilicon.com/kb/content_images/1/vince/zh-admin-install.gif


上載於步驟1取得的中文語言包 (直接上載 zip file 即可)
筆者為教學用途，同時安裝了前台和後台的中文語言

安裝完畢後，現在 Joomla 己經具備預設的英文和新安裝的繁體中文，您可以為網站設定那一種是"預設語言" (即客戶首次訪問網站所看到的語言)

2.2 - 登入後台 > Extensions > Language Manager

在 tab > "Site"
選 "繁體中文"，然後按右上方的 "Default" 按鈕，把中文設定為前台預設語言。

在 tab > "Administrator"
選 "繁體中文"，然後按右上方的 "Default" 按鈕，把中文設定為後台預設語言。

http://www.hksilicon.com/kb/content_images/1/vince/zh-admin-site.gif



後台設定成中文後，立刻便能看到效果，整個界面已經是熟識的中文了。

http://www.hksilicon.com/kb/content_images/1/vince/zh-admin-default-2.gif



我們再到前台看看，中文和英文版並沒有大分別，因為即使是界面改了中文，
網站的"內容"仍然是英文，我們需要為內容進行手動翻譯。

自行更改中文界面用字

如有些文字您覺得中文用詞不貼切 (如香港和台灣的用詞會有所出入)，您可以自行修改相關語言檔（用Notepad修改即可）：

  • 前台： /administrator/language/zh-TW/*.ini
  • 後台： /language/zh-TW/*.ini

中文化注意事項

1) 在新增或修改文章時，"發佈結束" 一欄會自動設定為當天日期，這會使新增的文章不能發佈。

http://www.hksilicon.com/kb/content_images/1/vince/zh-admin-never.gif



此問題的原因是 Joomla interpret 中文翻譯 "從不”(Never) 時發生問題。
解決方法十分簡單，修改:

/administrator/language/zh-TW/zh-TW.com_content.ini b (around line 98)
把:
NEVER=從不

改為：
NEVER=Never
