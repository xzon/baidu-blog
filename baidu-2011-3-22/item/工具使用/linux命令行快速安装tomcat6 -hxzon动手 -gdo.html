linux命令行快速安装tomcat6 -hxzon动手 -gdo

hxzon:ok。但是每次重新链接都要重复上述命令。http://XXX:8080/可远程进入首页。


1.放到/usr/local目录下，解压
tar xzvf apache-tomcat-6.0.10.tar.tar
装好后试着运行
[root@neudiadb ~]# /usr/local/tomcat/bin/startup.sh
Neither the JAVA_HOME nor the JRE_HOME environment variable is defined
At least one of these environment variable is needed to run this program

2.设置Java环境变量
cd /etc vi profile export JRE_HOME=/usr/java/jre1.6.0_07
运行仍然出现上面的错误。
3.使环境变量生效（在修改了PATH值或任何环境变量后，都要用export将其输出，新的值才能生效。）
export PATH
export JRE_HOME
此时运行/usr/local/tomcat/bin/shutdown.sh
/usr/local/tomcat/bin/startup.sh
Using CATALINA_BASE:   /usr/local/tomcat
Using CATALINA_HOME:   /usr/local/tomcat
Using CATALINA_TMPDIR: /usr/local/tomcat/temp
Using JRE_HOME:       /usr/java/jre1.6.0_07
成功。t_T
4.查看环境变量可以用这个命令
echo $PATH

安装jre
1.放到目录下。
2.更改文件权限，使可以执行
chmod a+x jre-6u<version>-linux-i586.bin
3.启动安装过程
./jre-6u<version>-linux-i586.bin
出现许可协议，按空格显示下一页，输入yes继续安装。
看到Done表示安装完成，会安装在/usr/java/jre1.6.0_<version>目录中。
4.如果是rpm安装，则启动安装
./jre-6u<version>-linux-i586-rpm.bin
输入yes后生成jre-6u<version>-linux-i586.rpm 文件。
再输入rpm -iv jre-6u<version>-linux-i586.rpm 完成安装。
