ITOT：基于TCP的ISO传输服务（ISO Transport Over TCP/IP）

http://www.networkdictionary.net/protocols/itot.php 

基于 TCP 的 ISO 传输服务（ITOT）是一种使 ISO 应用程序能够被移植到 TCP/IP 网络的机制。当要实现 ISO 应用程序到 TCP/IP（和 IPv6）环境的移植操作时，可以采取两种基本途径。其一是独立移植每个单个程序，在 TCP 上开发本地协议；其二是以在 TCP/IP 上分层 ISO 传输服务的观念为基础，这种方法为使用 ISO 传输服务的所有应用程序解决了问题。

ITOT 是一种传输服务，它等同于由 ISO 传输服务定义 [ISO8072] 提供的服务和接口，但它是在 TCP/IP 上实施 ISO 传输协议 [ISO8073]，而不是在 ISO 网络服务 [ISO8348] 上。TCP 端口 102 是为执行 ITOT 协议的主机预留的。

ITOT 中定义的两个变量是：“TCP 分类 0”和“TCP 分类 2”，以 ISO 传输分类 0 和分类 2 协议为基础。分类 0 提供了协商建立连接、分割传输数据以及协议错误报告所需的功能。它为传输连接提供了基于 NS —供应商 TCP 的流控制，并提供了基于 NS —供应商断开的传输断开。分类 0 适用于没有明确的传输断开的数据传输。

分类 2 也提供了协商建立连接、分割传输数据以及协议错误报告所需的功能。它为传输连接提供了基于 NS —供应商 TCP 的流控制，并提供了明确的传输断开。分类 2 适用于需要独立的标准和快速的数据通道或明确的传输断开的情况。
