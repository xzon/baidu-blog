7标点符号
标点符号一般是用来分隔语句或者表达式的，但是，有的标点符号也参与表达式运算，比如括号。标点符号的文法为：
punctuator : one of

! % ^ & * ( ) + = { } | ~
[ ] \ ; ' : " < > ? , . / #

在词法分析中，一般来说，对标点符号和操作符如下处理：
l        一元符号（单符号）直接返回。
l        多元符号根据名称返回。
