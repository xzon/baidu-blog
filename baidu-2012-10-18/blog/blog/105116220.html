javascirpt操纵jsp中变量 -jscode

原则：
jsp的变量在哪儿？在jvm里面
javascript的变量在哪儿？在浏览器的脚本引擎里面
jsp->javascript，只能通过写语言文本
javascript->jsp，只能通过表单或者query string
把概念搞清楚

推荐 黄立行——《彩虹的轨道》

方法一：
<%
String sha="123456";
%>
<script language="javascript">

var i="<%=sha%>";
window.alert(i);
window.location.href("xinximemo.asp?aisi="+i+"");

</script>
