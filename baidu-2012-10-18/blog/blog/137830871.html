hibernate查不到数据 -hxzon出错

public List getByArticle(String articleId) {
   String hql = "from Upload as p where p.articleId = '" + articleId
     + "' ";
   return uploadDao.getByHql(hql);
}

数据库明明有数据，可是就是查不到。后来发现，articleId插入到hql语句时，由于是字符串，必须加单引号，而这里出了错，因为为了好看，我把单引号和hql拼接的双引号写分开了，也就是加了一个空格，这是这个空格，被当作了articleId的一部分，使得查不到数据。
