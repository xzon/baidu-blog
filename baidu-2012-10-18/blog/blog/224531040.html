Source Code Structure
The table below lists the packages in the JIDE Common Layer. All packages are in jide-oss-
<version>.jar.

Packages Description

  1. com.jidesoft.swing   Common components.
  2. com.jidesoft.icon Icon related classes
  3. com.jidesoft.comparator Various Comparators. They all implement interface java.util.Comparator. ObjectComparatorManager provides a central place to register those comparators.
  4. com.jidesoft.converter Various ObjectConverters which can convert an object to/from String. ObjectConverterManager provides a central place to register those converters.
  5. com.jidesoft.grouper Various ObjectGroupers which can group several values into a named group. ObjectGrouperManager provides a central place to register those groupers.
  6. com.jidesoft.popup Popup component
  7. com.jidesoft.animation Animation related classes
  8. com.jidesoft.hints IntelliHints related classes

A general comment on our naming convention: If the class is modified from or based on an existing Swing/AWT class, we prefix the original Swing/AWT class name with Jide - for example, JideTabbedPane (you can tell that it is based on JTabbedPane from the name). If it’s a completely new component that doesn’t exist in Swing/AWT then we don’t prefix anything - for example, Calculator etc.
We will add more and more components to JIDE Common Layer in the future and we will keep the same package organization. If the component is complex enough or there are a group of components which share a common feature, there will be a separate package for it. If it is a very small component, we probably will put it under com.jidesoft.swing.

List of Components

Enhanced Swing Components or Classes

  1. JideButton: built on top of JButton with its ComponentUI. Best used on JToolBar or CommandBar( Part of in JIDE Action Framework. )
  2. JidePopupMenu: built on top of JPopupMenu. It will make sure the content of the popup menu to be inside the screen boundary. If the popup menu is very long, it will add scroll button to the top and bottom so that user can scroll it up and down.
  3. JideMenu: built on top of JMenu to allow lazily creation of menu items and to allow specifying the popup menu’s alignment.
  4. PartialLineBorder: built on top of LineBorder to only paint lines on certain sides.
  5. PartialEtchedBorder: built on top of EtchedBorder to only paint etched lines on certain sides.
  6. JideScrollPane: built on top of JScrollPane to support RowFooter, ColumnFooter as well as new corner components on either side of the scroll bars.
  7. SimpleScrollPane: built on top of JScrollPane to use four scroll buttons to do the scrolling. It has no scroll bar.
  8. JideSplitPane: A split pane that supports multiple split.
  9. JideTabbedPane: built on top of JTabbedPane to support many different tab styles, tab resize mode, tab leading component, tab trailing component etc.
  10. JideBoxLayout: similar to BoxLayout but can support three different constraints to give child component different resize weight.
  11. JideBorderLayout: built on top of BorderLayout just to make the north and south component the same width as center component.
  12. AutoResizingTextArea: A JTextArea which resizes vertically when there are more text in it.
  13. LabeledTextField: A JTextField which supports JLabel in front of it.
  14. MultilineLabel: A JTextArea that looks JLabel but supports multiple lines.
  15. RangeSlider: A JSlider that supports two thumbs to specify a range.
  16. StyledLabel: built on top of JLabel to support different styles on the text.
  17. TristateCheckBox: a check box that has three states.
  18. StandardDialog: built on top of JDialog to support common used dialog standards.

New Components

  1. JidePopup: support any popup window.
  2. ButtonPanel: support a panel for buttons in an OS-aware way.
  3. CheckBoxList/CheckBoxTree: A JList/JTree supports check box in the row/tree node.
  4. Calculator: a component for calculator.
  5. FolderChooser: a component for choosing folder.
  6. TitledSeparator: a JSeparator-like component with a title.

Usability enhancement

  1. AutoCompletion (JComboBox, JTextComponent)
  2. Resizable (ResizablePanel, ResizableWindow)
  3. Searchable (ComboBox, List, Table, TextComponent etc)
  4. SearchableBar: A searching component similar to Firebox searching bar. Built on top of Searchable.
  5. IntelliHints: provides dynamic hints to help user typing. It could be considered as an enhancd version of AutoCompletion. A typical use case for it is IntelliSense in editors of most IDEs.
  6. Overlayable: provides putting an overlay component on top of another component to display useful information such as validation error, process indicator etc.

Utilities

  1. SelectAllUtils
  2. Sticky
  3. ColorUtils
