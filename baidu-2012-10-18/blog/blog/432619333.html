9款流行的 HTML5/CSS3 框架推荐

 

HTML5 和 CSS3 框架越来越流行，紧跟潮流，本文推荐9款不错的HTML5 和 CSS3 框架。

 

Gridless

Gridless 是个很棒的HTML5 和CSS3 框架，制作跨平台网站排版、布局等。

Inuit.css

作者 Harry Roberts ，该框架应用于较小的屏幕，主要特点是甚至对IE6+的新的HTML5 元素的基本支持。

Less Framework 4

Less Framework 4 是个CSS网格系统，用于设计自适应网站，包括4个布局和3组排版预设。

Baseline

Baseline 用于快速进行网站原型设计，基线网格作为设计基础。

Gravity

Gravity 方便制作强大的、易维护的HTML5站点，帮助快速原型设计。

G5 Framework

G5 Framework 封装了很多的功能组合，见上图。

Perkins

Perkins 是个纯粹的HTML5/CSS3框架，有简单的HTML5模板，包括大部分标签、一组LESS样式，创建常见的任务如导航、圆角、渐变等。

52framework

52 Framework 主要用于优化HTML5和CSS3的跨浏览器兼容性的框架。

HTML5 Framework

HTML5 Framework 恰如其名，在HTML5创建布局的框架。基本文件包括HTML5元素，并有一CSS文件风格化这些元素。
