判断素数-算法设计-C源代码 -good code

判断m是否素数。我们采用的算法是这样的:让m被2到sqrt(m)除，如果m能被2～sqrt(m)之中的任何一个整数整除，则提前结束循环，此时i必然小于或等于sqrt(m)；如果m不能被2～sqrt(m)之间的任一整数整除，则退出循环后i大于sqrt(m)。最后判断i是否大于sqrt(m)，就能判断m是否为素数了。程序如下：

#include <math.h>
#include <stdio.h>

void main()
{
   int m,i,k;
   printf("输入一个正整数：");
   scanf("%d",&m);
   k=(int)sqrt(m);
   for(i=2;i<=k;i++)
     if(m%i==0) break;
   if(i>k) printf("%d是一个素数。\n",m);
   else printf("%d不是一个素数。\n",m);
}
